A1. first accepted v2 SIGHUP live reload
- Start node with baseline trust bundle.
- Candidate bundle + v2 sidecar are valid and higher/effective.
- Send real SIGHUP.
- Expected:
  - SIGHUP trigger observed.
  - v2 path selected.
  - Run 130 v2 verifier succeeds.
  - v2 marker decision accepts first write.

  - v2-after-v1 migration accepted.
  - marker becomes V2.
