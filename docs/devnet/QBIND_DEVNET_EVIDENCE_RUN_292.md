## 6. Schema/domain binding release evidence

The release helper exercises the Run 291 schema/domain binding on real RocksDB: it opens a database, writes the schema marker and domain metadata, and asserts fail-closed typed errors for `SchemaUnsupported`, `SchemaMarkerMalformed`/`SchemaMarkerMissing`, `MetadataMissing`/`MetadataMalformed`, and `DomainMismatch{field}` when a reopened database's identity (env / chain / genesis / seq) disagrees with the persisted domain metadata. `EventDomainMismatch{field}` is asserted when an event's domain disagrees with the backend's bound identity.

## 7. Durability/reopen release evidence

The helper writes typed replay records, calls `close_or_flush`, drops the backend, and reopens the same on-disk directory via `open_or_initialize`, asserting `read_replay_record`, `scan_replay_records`, and `recover_replay_window` return the persisted records across reopen — proving real on-disk durability rather than in-memory state. Atomic `WriteBatch` writes are exercised, and `record_replay_event_simulate_precommit_failure` / `record_replay_event_simulate_partial_stage_failure` prove that a simulated precommit/partial-stage failure leaves no partially applied record (`PartialResidueDetected` fails closed on reopen).

## 8. Idempotency/equivocation release evidence

Re-recording an identical replay record is idempotent (no second record, no error). Recording the same record id with a different digest fails closed as `Equivocation`. `Observed → Consumed` stage ordering is enforced: a `Consumed` event without the required prior `Observed` digest, or with a mismatched prior digest, fails closed with `OrderingViolation`; malformed events fail closed with `MalformedEvent`.

## 9. Corruption/wrong-domain fail-closed release evidence

The helper drops the typed backend, reopens the raw `rocksdb::DB`, and overwrites a record key (`durable_replay_rocksdb_record_key`) with corrupt bytes and with wrong-domain bytes. On the next typed read the backend fails closed with `CorruptRecord`, `CorruptDigest{record_id}`, or `DomainMismatch{field}` — never returning a silently substituted or in-memory value. `PartialResidueDetected` covers partial-write residue.
